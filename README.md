# 🎬 VIP Кино Сувгийн Telegram Бот

## 📁 Файлын бүтэц

```
vip_bot/
├── bot.py           ← Ботын үндсэн код
├── database.py      ← SQLite өгөгдлийн сан
├── config.py        ← Тохиргоо
├── requirements.txt ← Python сангууд
├── Dockerfile       ← Docker image
├── docker-compose.yml
├── .env.example     ← Тохиргооны загвар
└── README.md
```

---

## ⚙️ Тохиргоо

### 1. `.env` файл үүсгэх

```bash
cp .env.example .env
```

`.env` файлыг нээж өөрийн утгуудыг оруулна:

```
BOT_TOKEN=ботын_токен
ADMIN_ID_1=1-р_админы_telegram_id
ADMIN_ID_2=2-р_админы_telegram_id
VIP_GROUP_ID=vip_группийн_id
```

### 2. Telegram ID олох

- **Ботын токен**: [@BotFather](https://t.me/BotFather) → `/newbot`
- **Telegram ID**: [@userinfobot](https://t.me/userinfobot) руу `/start` бич
- **Группийн ID**: Группт [@RawDataBot](https://t.me/RawDataBot) нэмж мессеж бич

---

## 🚀 Ажиллуулах

### Docker ашиглан (санал болгосон):

```bash
docker-compose up -d
```

### Шууд Python ашиглан:

```bash
pip install -r requirements.txt
python bot.py
```

## 📋 Командуудын жагсаалт

### Хэрэглэгчийн мессеж

| Үйлдэл | Тайлбар |
|--------|---------|
| Анх удаа бичих | Автомат хариулт авна |
| Дараагийн мессежүүд | Админд дамжуулагдана |

### Админы командууд

| Команд | Тайлбар |
|--------|---------|
| `/reply <id> <текст>` | Хэрэглэгчид хариулах |
| `/addvip <id> <хоног>` | VIP нэмэх |
| `/extendvip <id> <хоног>` | VIP сунгах |
| `/removevip <id>` | VIP цуцлах |
| `/viplist` | Бүх VIP жагсаалт |
| `/vipinfo <id>` | Хэрэглэгчийн дэлгэрэнгүй |
| `/stats` | Ерөнхий статистик |
| `/setreply` | Автомат хариулт өөрчлөх |
| `/viewreply` | Одоогийн автомат хариулт |

## 💬 Автомат хариулт өөрчлөх

Админ `/setreply` бичнэ:

```
Та: /setreply
Бот: ✏️ Шинэ автомат хариулт текстийг бичнэ үү...
Та: 🔥 Шинэ кинонууд нэмэгдсэн! VIP авах бол холбогдоно уу.
Бот: ✅ Амжилттай хадгалагдлаа!
```

Одоогийн хариултыг харах: `/viewreply`
